# 🚑 ResQTrack — Real-Time Disaster Resource Management System

A full-stack system for managing rescue teams, tracking resources, and enabling real-time disaster response coordination.

---

## 🚀 Features

- Central Command Dashboard
- Rescue Team Login (Key-Based)
- Real-time updates using WebSocket
- Inventory management system
- Low resource alerts
- AI chatbot (planned)

---

## ⚡ Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: MongoDB Atlas
- Real-time: WebSocket

---

## ⚙️ Setup Instructions

1. Install dependencies:
2. npm install


2. Run server:

npm start


3. Open in browser:

http://localhost:3000


---

## 📡 API Endpoints

- GET `/api/initial-data`
- POST `/api/login`
- POST `/api/deploy`
- DELETE `/api/delete-team/:id`

---


## 💡 Description

ResQTrack helps improve disaster management by enabling real-time coordination between rescue teams and command centers.
